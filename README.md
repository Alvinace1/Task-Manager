# Task Manager App 🚀

A simple and interactive **Task Manager** built with **React**.  
Users can add, delete, complete, and filter tasks. Tasks are saved in the browser, so your data persists on refresh.

---

## Features

- Add new tasks 📝
- Delete tasks ❌
- Mark tasks as completed ✅
- Filter tasks:
  - All  
  - Completed  
  - Pending  
- Clear all tasks at once 🗑️
- Press **Enter** to add a task (keyboard-friendly)
- Tasks persist using `localStorage`
- Clean and responsive UI

---

## Technologies Used

- **React** – for building the UI  
- **CSS** – for styling  
- **localStorage** – to persist tasks in the browser
